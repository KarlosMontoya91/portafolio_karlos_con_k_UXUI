# KRM Portfolio (futurista)

## Requisitos
- Node.js 18+

## Instalar
```bash
npm install
npm run dev
```

## Personaliza
Edita `src/data/profile.ts`:
- links (GitHub/LinkedIn/Behance/WhatsApp)
- proyectos, tags y highlights

## Deploy
- Vercel/Netlify: build command `npm run build` y output `dist`.
